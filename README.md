# Docker TALL Stack 开发环境

基于 Docker 的 PHP 全栈开发环境，整合 TALL Stack (Tailwind CSS, Alpine.js, Laravel, Livewire)。

## 特性

- 🚀 完整的 TALL Stack 支持
- 🔧 开箱即用的 Docker 环境
- 🐛 集成 Xdebug 调试
- 🔄 支持热重载开发
- 📦 Redis 缓存支持
- 📊 MySQL 数据库
